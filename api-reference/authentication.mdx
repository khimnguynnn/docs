title: "Xác thực API"
description: "Cách xác thực các yêu cầu tới API"
---

Mỗi request tới API phải được xác thực bằng API hợp lệ. Bạn có thể tìm thấy cả khóa API public và API secret của mình trong [API settings](https://app.useplunk.com/settings/api) dự án của bạn.

```bash
'Authorization': 'Bearer <team_token>'
```

## Public API Key


| Action                       | Allowed |
|------------------------------|---------|
| Theo dõi sự kiện             | ✔️      |
| Gửi transactional emails     | ✖       |
| Truy xuất contacts           | ✖       |
| Tạo contacts mới             | ✖       |
| Xóa contacts                 | ✖       |
