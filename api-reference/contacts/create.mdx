---
title: "Create a contact"
api: "POST https://api.useplunk.com/v1/contacts"
description: "Create a new contact"
---
<Warning>
    In most cases, it is recommended to use the [track](/api-reference/actions/track) endpoint to create a contact.
    When a new contact tracks an event Plunk will automatically create the contact, thus saving you an API call.
</Warning>


### Body
<ParamField body="email" type="string" placeholder={'hello@useplunk.com'} required>
    The email address of the contact.
</ParamField>
<ParamField body="subscribed" type="boolean" placeholder={'hello@useplunk.com'} required>
    Whether the contact is subscribed to marketing emails.
</ParamField>
<ParamField body="data" type="object" placeholder={{"project": "Plunk"}}>
    Metadata to store with the contact.
</ParamField>

### Headers
<ParamField header="Content-Type" type="string" placeholder={"application/json"} required>
    `application/json`
</ParamField>
